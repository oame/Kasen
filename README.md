# Kasen

Image Assistant for Coda 2

# Requirements

* Coda 2
* Ruby
* ImageMagick(Recommend: Homebrew package)

# How to use

### Measuring image size

`Kasen => Measure Image Size` to measuring image size and insert `width` and `height`.

shortcut key is `Command + Shift + M`.
